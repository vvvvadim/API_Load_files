# Система загрузки файлов API

Современное веб-приложение для загрузки и управления файлами с акцентом на фотографии и управление заказами.

## Возможности

- **Современный пользовательский интерфейс**
  - Адаптивный дизайн для работы на компьютерах и мобильных устройствах
  - Чистый и интуитивно понятный интерфейс загрузки файлов
  - Мгновенная обратная связь при выборе файлов
  - Стилизованные кнопки и элементы формы

- **Управление файлами**
  - Поддержка загрузки нескольких файлов
  - Проверка форматов изображений
  - Отображение статуса выбора файлов в реальном времени
  - Предварительный просмотр списка файлов перед загрузкой

- **Управление заказами**
  - Отслеживание номеров заказов
  - Организованное хранение файлов по номерам заказов
  - Безопасная обработка файлов

## Технический стек

- **Серверная часть**
  - Сервер на Python
  - Фреймворк FastAPI
  - Надежная обработка ошибок
  - Эффективная обработка файлов

- **Клиентская часть**
  - HTML5
  - CSS3 с современными возможностями
  - JavaScript для динамических взаимодействий
  - Адаптивный дизайн с медиа-запросами
  - Интеграция пользовательского шрифта (Open Sans Condensed)

## Требования

- Docker
- Docker Compose

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone [адрес-репозитория]
```

2. Перейдите в директорию проекта:
```bash
cd [название-директории]
```

3. Запустите приложение с помощью Docker Compose:
```bash
docker-compose build
docker-compose up -d
```

Приложение будет доступно по адресу: http://localhost:8000

## Остановка приложения

Для остановки приложения выполните:
```bash
docker-compose down
```

## Структура проекта

Проект организован следующим образом:

```
app/
├── main.py          # Основная точка входа приложения
├── media.py         # Функциональность обработки медиафайлов
├── orders.py        # Система управления заказами
└── ui/
    └── index1.html  # Основной пользовательский интерфейс
```

## Использование

1. Откройте веб-интерфейс в браузере по адресу http://localhost:8000
2. Введите или выберите номер заказа
3. Нажмите "Обзор" для выбора файлов
4. Выберите одно или несколько изображений
5. Нажмите "Отправить" для загрузки файлов
